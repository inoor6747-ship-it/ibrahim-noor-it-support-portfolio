# IT Support Lab Portfolio

Hands-on troubleshooting practice labs simulating real Help Desk and Junior IT support tickets.

Live site:
https://inoor6747-ship-it.github.io/ibrahim-noor-it-support-portfolio/

## What this demonstrates
- Windows troubleshooting methodology
- Active Directory user and permission management
- Network diagnostics using command-line tools
- Ticket documentation from report to resolution

## Labs
- Windows Troubleshooting
- Active Directory Access Issues
- Ticket Simulation (printer, VPN, user issues)
- Network Diagnostics

Each scenario follows a real support workflow:
Problem → Investigation → Root Cause → Resolution → Verification
